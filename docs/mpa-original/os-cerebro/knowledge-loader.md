## Quando carregar

- CoS nunca carrega diretrizes profundas para decidir rota.
- Especialista carrega no maximo uma diretriz primaria.
- Gate carrega apenas a diretriz da falha se precisar revisar.
- Diretriz de operadora especifica so entra quando a task envolve aquela operadora.
- Diretriz de regulamentacao ANS so entra quando ha risco regulatorio identificado.

## Quando nao carregar

- Tarefa de ledger, renomeacao, resumo ou organizacao simples.
- Pergunta factual simples que o context pack ja responde.
- Ajuste de formato sem mudanca de conteudo.
- Quando o context pack ja contem a informacao necessaria.
- Quando a task e de um departamento que nao precisa daquela diretriz.
