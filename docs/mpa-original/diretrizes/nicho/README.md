# Nicho

Arquivos de nicho sao mapas profundos do segmento. Nao contem dados de cliente especifico nem templates de copy.

## Arquivo disponivel

- `corretora-saude.md` — mapa completo do nicho de corretora de planos de saude: dores, desejos, objecoes, ciclo de venda, riscos regulatorios, metricas e provas

## Como usar

1. Comando MPA detecta que a task envolve estrategia do negocio ou posicionamento
2. Carrega `corretora-saude.md` como contexto
3. Usa como mapa de restricoes, dores comuns, provas tipicas e riscos
4. Nao substitui VOC real — complementa

## Quando carregar

- Planejamento estrategico da corretora
- Argumentacao sobre por que usar o MPA
- Criacao de conteudo sobre a profissao de corretor
- Analise de concorrencia
- Definicao de posicionamento
- Quando a corretora pergunta "como resolver X no meu negocio"
