# Design — Diretrizes Visuais para Corretora de Saude

> Principios visuais para todo material produzido pela MPA: posts, carrosseis, propostas, comparativos, apresentacoes, stories e materiais de venda. Design nao e decoracao — e comunicacao visual que reforca posicionamento e gera confianca.

---

## Principio central

Todo material visual da corretora precisa transmitir tres coisas:

1. **Confianca** — parece profissional, nao amador
2. **Clareza** — entende em 3 segundos o que e
3. **Acessibilidade** — qualquer pessoa entende, sem jargao visual

Design de corretora de saude nao e design de startup, nao e design de moda, nao e design de fast food. E design de **confianca** — o lead precisa sentir que pode confiar a saude da familia a essa empresa.

---

## Identidade visual

### Elementos obrigatorios

| Elemento | Diretriz |
|---|---|
| Logo | Sempre presente, proporcional, nao distorcido |
| Nome | "[Nome da Corretora]" ou "[Nome da Corretora]" — nunca abreviado sem contexto |
| Contato | WhatsApp e/ou telefone visivel em todo material de divulgacao |
| Tom | Profissional, acolhedor, humano — nunca corporativo frio |

### Paleta de cores

A paleta deve transmitir saude, confianca e profissionalismo:

| Uso | Direcao | Evitar |
|---|---|---|
| Cor primaria | Tons que transmitem confianca e saude (azul, verde, azul-petroleo) | Vermelho agressivo, amarelo neon |
| Cor secundaria | Complementar quente para acolhimento (dourado, coral suave, verde-claro) | Preto pesado como dominante |
| Fundo | Branco, off-white ou tons claros — limpeza | Fundos escuros como padrao (ok para destaque) |
| Texto | Escuro sobre claro — maximo contraste | Cinza claro sobre branco (ilegivel) |
| Destaque | Cor primaria para CTAs e informacoes-chave | Multiplas cores competindo |

### Tipografia

| Uso | Direcao |
|---|---|
| Titulos | Fonte sem serifa, peso bold, tamanho grande, legivel no celular |
| Corpo | Fonte sem serifa, peso regular, tamanho confortavel (min 14px em digital) |
| Destaques | Mesmo tipo, peso semi-bold ou cor diferente |
| Evitar | Fontes decorativas, cursivas em textos longos, mais de 2 familias |

## Materiais da corretora

### Post Instagram (feed)

| Aspecto | Diretriz |
|---|---|
| Formato | 1080x1080 (quadrado) ou 1080x1350 (4:5) |
| Texto no visual | Maximo 30% da area — Instagram penaliza texto excessivo |
| Headline | 5-8 palavras, legivel sem zoom |
| Corpo | Informacao util, educativa — nao panfleto |
| CTA | Claro e direcionado: "Fale comigo", "Salve esse post", "Comente X" |
| Marca | Logo discreto, nao dominante |
| Imagens | Pessoas reais > stock generico. Diversidade de idade e familia |

### Carrossel Instagram

| Aspecto | Diretriz |
|---|---|
| Slides | 5-10 slides, uma ideia por slide |
| Capa | Gancho forte — motivo para deslizar |
| Fluxo | Cada slide conecta ao proximo — nao sao soltos |
| Ultimo slide | CTA + contato + identidade |
| Tipografia | Grande o suficiente para celular, contraste alto |
| Informacao | Educativa: "3 coisas que ninguem te conta sobre plano de saude" |

### Stories

| Aspecto | Diretriz |
|---|---|
| Formato | 1080x1920 (9:16) |
| Duracao | Sequencia de 3-7 stories |
| Tom | Bastidores, humanizado, conversa |
| Interacao | Enquete, caixa de perguntas, quiz — gerar engajamento |
| Venda | Stories e para criar relacionamento, nao panfleto. Vender no maximo 20% do tempo |
| Texto | Curto, grande, legivel rapido |

### Reels

| Aspecto | Diretriz |
|---|---|
| Formato | 1080x1920 (9:16) |
| Duracao | 15-60 segundos (ideal: 30s) |
| Gancho | Primeiros 2 segundos decidem se a pessoa continua |
| Conteudo | Dica pratica, mito vs. verdade, bastidores, depoimento |
| Legenda | Texto na tela — maioria assiste sem som |
| CTA | Ultimo segundo ou na legenda |

### Comparativo de planos

| Aspecto | Diretriz |
|---|---|
| Formato | PDF ou imagem horizontal legivel no celular |
| Estrutura | Tabela comparativa: operadora x operadora, item por item |
| Informacoes | Valor, cobertura, rede, carencia, coparticipacao, acomodacao |
| Destaque | Recomendacao da corretora com justificativa |
| Rodape | Validade da cotacao, data da tabela, contato |
| Fonte dos dados | Tabela vigente da operadora — nunca valores inventados |
| Clausula | "Valores sujeitos a confirmacao pela operadora. Cotacao valida por X dias" |

### Proposta formal

| Aspecto | Diretriz |
|---|---|
| Formato | PDF profissional |
| Capa | Logo + nome do lead/empresa + data |
| Conteudo | Dados do lead, plano recomendado, coberturas, valores, carencias, rede |
| Tom | Profissional, claro, sem jargao desnecessario |
| Proximos passos | Claro: "Para prosseguir, me envie [documentos]" |
| Legal | Clausula de validade, fonte dos valores, aviso sobre confirmacao de cobertura |

### Apresentacao comercial

| Aspecto | Diretriz |
|---|---|
| Formato | Google Slides ou PDF, max 10-15 slides |
| Estrutura | Problema -> Solucao -> Opcoes -> Recomendacao -> Proximos passos |
| Visual | Limpo, pouco texto por slide, graficos quando aplicavel |
| Dados | Sempre com fonte (ANS, operadora, pesquisa) |
| Personalizacao | Adaptar para o perfil do lead (PF, PJ, quantidade de vidas) |

## Hierarquia visual

### Prioridade de informacao

1. **O que e** — titular, headline, contexto
2. **Por que importa** — beneficio principal, dor que resolve
3. **O que fazer** — CTA, proximo passo
4. **Quem fala** — logo, contato, credencial

Nunca:

- Logo maior que o conteudo
- CTA escondido ou ambiguo
- Informacao tecnica antes do beneficio
- Multiplas mensagens competindo no mesmo espaco

## Imagens

### O que usar

- Pessoas reais — familias, profissionais, idosos, criancas
- Diversidade de faixa etaria (o publico de plano de saude vai de 0 a 80+)
- Ambientes acolhedores — consultorio, hospital moderno, casa
- Expressoes de tranquilidade, seguranca, cuidado
- Fotos da propria corretora ([Nome]) quando possivel — humaniza

### O que evitar

- Stock generico de "pessoa sorrindo com laptop"
- Imagens de hospital frio, equipamento medico assustador
- Fotos de banco de imagens facilmente reconheciveis como fake
- Imagens que transmitem doenca, sofrimento, medo (plano e sobre protecao, nao sobre medo)
- Fotos com marca d'agua
- Imagens sem direito de uso

## Acessibilidade

| Aspecto | Diretriz |
|---|---|
| Contraste | Texto sempre legivel sobre fundo — minimo ratio 4.5:1 |
| Tamanho de fonte | Minimo 14px digital, 10pt impresso |
| Alt text | Descrever imagens em posts para acessibilidade |
| Daltonismo | Nao depender so de cor para transmitir informacao |
| Celular | Todo material deve ser legivel na tela do celular, nao so no desktop |
| Linguagem visual | Icones e simbolos universais quando possivel |

---

## Compliance visual

| Regra | Detalhe |
|---|---|
| Valores | Nunca colocar valores de plano em material publico sem clausula de validade e fonte |
| Coberturas | Nunca prometer cobertura especifica em imagem — pode mudar por plano/operadora |
| Operadoras | Se usar logo de operadora, verificar permissao e uso correto |
| ANS | Se citar regulacao, citar corretamente (numero da RN, nao generalizar) |
| Depoimentos | Se usar depoimento de cliente, ter autorizacao (LGPD) |
| Antes/depois | Nao usar formato "antes/depois" que sugira resultado medico |

## Anti-patterns visuais

| Anti-pattern | Por que e ruim |
|---|---|
| "Mega oferta!!! Plano a partir de R$99!!!" | Parece fraude. Precos dependem de faixa/regiao. Propaganda enganosa |
| Muitas cores, muitas fontes, muitos elementos | Confuso, amador, nao transmite confianca |
| Copiar visual de outra corretora | Perde identidade, pode ter problemas legais |
| Material so com texto, sem hierarquia | Ninguem le. Bloco de texto afasta |
| Material lindo mas sem CTA | Bonito e inutil se o lead nao sabe o que fazer depois |
| Informacao desatualizada | Pior que nao ter material. Valor errado = problema regulatorio |
| Usar Canva template generico sem adaptar | Parece que qualquer um fez. Nao transmite personalidade |

---

## Teste final de design

Para cada material, verificar:

- [ ] Transmite confianca em 3 segundos?
- [ ] A informacao principal esta clara sem zoom?
- [ ] Funciona no celular (nao so no desktop)?
- [ ] CTA esta visivel e claro?
- [ ] Nao tem informacao incorreta ou desatualizada?
- [ ] Nao promete cobertura ou valor sem clausula?
- [ ] Tom visual e coerente com a marca?
- [ ] Pessoa leiga entenderia sem ajuda?
