O que significa "Recorrente" na prática?

Quando você salvar uma conta como Recorrente, o que deve acontecer?

A

Substitui as Despesas Fixas

A seção de "Despesas Fixas" some. Tudo vira conta Recorrente na lista unificada. Você cadastra uma vez — o sistema gera automaticamente todo mês. Mais simples, tudo num lugar só.

B

Só um rótulo — não gera nada automático

"Recorrente" é apenas uma informação na conta (como categoria). As Despesas Fixas continuam funcionando igual. Mais simples de implementar, sem risco de quebrar o que já funciona.

C

Integra com Despesas Fixas

Ao marcar como Recorrente, a conta vira uma Despesa Fixa por baixo dos panos. A seção de Despesas Fixas permanece separada para gerenciar as recorrentes. Mantém o que existe, só muda o acesso.
